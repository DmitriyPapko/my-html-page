# My HTML Page

This project is a small vanilla JavaScript demo with accompanying tests. It includes example game logic and an HTML demo page.

## Requirements
- Node.js (v14 or later)

## Installation
```bash
npm install
```

## Running
Open the HTML file in the `docs/` directory in a browser.

## Testing
```bash
npm test
```

## Project structure
- `src/` - source code for the demo
- `test/` - test files
- `docs/` - HTML documentation and demo
